Stash

Clipboard history for macOS

macOS

About

Stash lives in your menu bar and quietly captures everything you copy — text, rich text, images, URLs, file paths — so nothing is ever lost.

Click the menu bar icon to browse your recent entries with content previews and image thumbnails. Click any entry to paste it, or right-click for quick actions like pin, delete, and full-size image preview.

Features

Text, rich text, images, URLs, files
AES-256-GCM encrypted storage
Password manager auto-filtering
Pin entries to keep them safe
Source app detection per entry
All data local — no cloud, no network
Configurable retention and auto-delete
Auto-update via Sparkle
