Phosphor

LED matrix display simulator for macOS and iOS.

About

Phosphor simulates a physical LED dot-matrix display on your screen. It renders text, images, and animations using a Metal-based renderer with authentic visual effects like phosphor persistence, dome highlights, and chromatic aberration.

Control the display over your local network via a built-in HTTP API, or use the web admin panel from any browser on the same network.

Features

  • Metal SDF renderer with per-dot glow
  • Scrolling text with adjustable speed and color
  • Full RGB per-pixel grid control
  • GIF animation playback
  • 12 visual effect toggles
  • Built-in web admin panel
  • HTTP API for local network control
  • Clock mode
