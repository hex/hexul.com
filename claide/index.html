Claide

A macOS companion app for Claude Code sessions

macOS 14+ · MIT License
This project is no longer actively maintained. The source code remains available on GitHub.

Terminal, issue tracker, dependency graph, and file monitor in one window. Claide sits alongside Claude Code and surfaces project context without switching apps.

Install

$ brew install --cask hex/tap/claide

Or download from GitHub Releases

Capabilities

Terminal GPU-accelerated rendering powered by Metal and Rust
Board Kanban view with beads issues and Claude Code tasks
Graph Interactive force-directed dependency visualization
Files Live monitoring of changes during sessions
Status Claude Code context window tracking

Swift 6 · Metal · Rust · 165 tests
