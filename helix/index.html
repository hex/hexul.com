Helix

A macOS menu bar app for mutagen file sync

macOS 14+ · MIT License

Session dashboard, conflict resolution, and endpoint management for mutagen. Helix wraps the CLI in a native SwiftUI interface so you can monitor, create, and edit sync and forwarding sessions from the menu bar.
