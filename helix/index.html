Helix

A native macOS menu bar app for managing mutagen file sync and network forwarding sessions.

macOS 14+ MIT License v2026.2.0
Download latest release View source

Requires mutagen installed on your system

Session dashboard with endpoints, configuration, and ignore rules

Capabilities

Session dashboard
Browse, search, and filter all sync and forward sessions in a three-column layout
Conflict resolution
Inspect per-file conflicts side by side, keep either side with one click
Create and edit
Step-by-step wizard for sync sessions, streamlined form for port forwarding
CLI preview
See the exact mutagen command before you run it, with a copy button
Menu bar status
Live icon reflects overall session health: healthy, syncing, conflicts, errors
Console log
Timestamped panel showing every CLI command, result, and error
Auto-update
Built-in Sparkle integration checks for updates from the menu bar
Keyboard shortcuts
Cmd+N new sync, Cmd+Shift+N new forward, Cmd+R refresh

Endpoints

Local
Any path on your Mac
SSH
Remote servers via OpenSSH
Docker
Containers on any host
